# AI Proctoring System

An AI-based online exam monitoring system designed to improve test integrity by tracking user behavior and suspicious activity.

## Features
- Face/activity monitoring
- Suspicious behavior detection
- Backend integration
- Real-time or session-based proctoring logic
- Alert/log generation

## Tech Stack
- Python
- Flask / backend
- HTML, CSS, JavaScript
- Computer vision / detection logic

## How It Works
1. User starts exam session
2. Camera/activity is monitored
3. Suspicious behavior is detected
4. System records or flags events
5. Admin/professor can review alerts
